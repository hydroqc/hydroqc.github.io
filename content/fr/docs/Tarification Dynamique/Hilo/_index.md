Pour une intégration plus complète de l'écosystème Hilo, de ses défis ainsi que de la consommation en temps réel, nous vous suggérons d'utiliser le projet d'intégration Home-Assistant [Hilo](https://github.com/dvd-dev/hilo).

Il est possible d'utiliser le projet Hydroqc lorsque vous être inscrit à [Hilo](https://www.hydroquebec.com/residentiel/mieux-consommer/offres-pour-economiser-cet-hiver/service-maison-intelligente-hilo.html). Dans ce cas vous récupérerez vos informations de compte de base disponible dans votre portail.
